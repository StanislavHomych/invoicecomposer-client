# Logo Files for Schema Indexing

Для правильної індексації schema потрібні наступні файли в цій директорії:

1. **logo.png** - основний логотип для schema.org (використовується в Organization та BlogPosting схемах)
   - Шлях доступу: `https://invoicecomposer.com/logo.png`
   - Рекомендований розмір: мінімум 112x112px, оптимально 512x512px

2. **favicon.ico** - іконка сайту (опціонально)
   - Шлях доступу: `https://invoicecomposer.com/favicon.ico`
   - Розмір: 16x16, 32x32, або 48x48px

**Важливо:**
- Всі файли з директорії `public/` автоматично копіюються Vite в корінь `dist/` під час білда
- Файл `logo.png` має бути з маленької літери для правильного доступу через schema
- Якщо у вас є `Logo.png` (з великої літери), створіть копію `logo.png` (з маленької літери)

**Як додати файли:**
1. Помістіть `logo.png` в цю директорію (`client/public/`)
2. (Опціонально) Помістіть `favicon.ico` в цю директорію
3. Після білда файли будуть доступні за `/logo.png` та `/favicon.ico`
